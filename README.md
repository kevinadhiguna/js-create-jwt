## How to Run

1) Clone this repository : <br/>
```
git clone https://github.com/kevinadhiguna/js-create-jwt
```

2) Install dependencies : <br/>
```
yarn
```
or
```
npm install
```

3) Generate a new JWT : <br/>
```
yarn generate
```
or
```
npm run generate
```
